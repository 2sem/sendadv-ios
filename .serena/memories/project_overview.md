## Purpose
An iOS application for sending SMS messages to multiple recipients. The app provides functionality to:
- Manage recipient lists and rules
- Filter and organize contacts
- Send bulk SMS messages
- Track and manage message sending

## Platform & Requirements
- **Minimum iOS Version**: iOS 13.0 (as per xcconfig files)
- **Target iOS Version**: iOS 18.0 (as per Project.swift)
- **Supported Devices**: iPhone and iPad
- **Swift Version**: 5.0
- **Xcode**: Compatible up to version 26.0

## Key Features
- Contact access and management
- Recipient list creation with filtering rules
- SwiftData-based persistence
- Google AdMob integration (Full, Launch, Native, Reward ads)
- Firebase integration (Crashlytics, Analytics, Messaging, RemoteConfig)
- Kakao SDK integration
- App Store review management
- App tracking transparency support
